# 🔥 Xây dựng Firewall (pfSense) và IDS (Suricata)

## 📌 Giới thiệu
Dự án mô phỏng hệ thống bảo mật mạng sử dụng pfSense và Suricata để phát hiện và ngăn chặn xâm nhập trái phép.

## 🛠 Công nghệ sử dụng
- pfSense 2.8.1
- Suricata IDS/IPS
- VMware
- Kali Linux
- Windows 10

## ⚙️ Chức năng đã triển khai
- Chặn ICMP theo IP
- Phát hiện quét port
- Ngăn chặn SYN Flood
- NAT cấu hình đầy đủ
- Rule firewall chi tiết
